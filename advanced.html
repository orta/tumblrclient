Making the complex possible and the simple easy
  1. orta:
    AppleScript support:
    Tumblg supports AppleScript, this means you can do something like

    tell application "Tumblg"
      post "hello world" as type "text"
    end tell

    and Tumblg will happliy go along and post your text using your settings. With this you could do anything, for example, the following script posts the current MP3 from iTunes and uses quicksilver to provide a comment.

    tell application "iTunes"
      if player state is playing then
        set theTrack to current track
        set songLocation to location of theTrack as string
      end if
    end tell
    tell application "Tumblor"
      post file songLocation with description "awesome"
    end tell

  2. orta:
    Feed Reader support:
    Tumblg supports importing from your favourite feed reader. This means you an simply hit the Post To Weblog button and Tumblg will grab the selected text and url and import it. You can choose which type of post you want to make by default in the preferences
